Purpose: Create an intelligent AI journal buddy that is personal to the user.

Setup: The user is prompted with a welcome or greeting message and we ask them a series of questions to inspire them to write.

Usage: It's an AI chatbot that is interactive with the user's responses.

Responsible AI Use: We will flag for certain key words like "suicide" or harmful messages. 

“Rule Behavior” section: 
inputs: name, how long I want to write for, mood, time of day and year.
example command: How long do you want to write for?
example output: inspirational quote
